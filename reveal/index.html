📜 A História da IA

70 anos em 10 minutos

🏛️ 1950–1970 — Os Pioneiros

1950 · Teste de Turing

Alan Turing publica "Computing Machinery and Intelligence" — propõe que se uma máquina conversa e não dá pra distinguir de um humano, ela "pensa". Até hoje é referência filosófica.

1956 · Conferência de Dartmouth

John McCarthy cunha o termo "Inteligência Artificial". 10 pesquisadores se reúnem por 2 meses com a ambição de resolver IA num verão. Spoiler: não conseguiram.

1958 · Perceptron

Frank Rosenblatt cria o primeiro neurônio artificial que aprende. A mídia declara: "a marinha criou um computador que pensa". Funcionava para classificações lineares simples.

1966 · ELIZA

Joseph Weizenbaum cria ELIZA no MIT — um chatbot que simulava um psicólogo usando pattern matching. Pessoas se abriram emocionalmente com ela, mesmo sabendo que era um programa. Primeiro "chatbot" da história.

❄️ 1969 — 1º Inverno da IA: Minsky publica livro provando limitações do Perceptron → funding secou, pesquisa parou.

⚙️ 1980–2000 — Sistemas Especialistas e Redes Neurais

1980s · Expert Systems

Empresas gastam bilhões em sistemas baseados em regras IF-THEN escritas por humanos. MYCIN diagnosticava infecções, XCON configurava computadores. Funcionavam, mas eram frágeis e caros de manter.

1986 · Backpropagation

Rumelhart, Hinton & Williams formalizam o algoritmo que permite treinar redes neurais profundas. É a base de TUDO que veio depois — até hoje o GPT é treinado com variantes de backprop.

1997 · Deep Blue vence Kasparov

IBM constrói um supercomputador que avalia 200 milhões de posições/segundo. Vence o campeão mundial de xadrez. Manchete mundial. Mas era "força bruta" — não aprendia, não generalizava.

1990s · N-grams e ML clássico

Estatística domina NLP: modelos prevêem a próxima palavra contando co-ocorrências. SVM, Random Forest, boosting resolvem problemas práticos. Mas cada tarefa precisa de features manuais.

❄️ 2º Inverno (late 80s–90s): Expert Systems falham em escalar → novo corte de funding. Redes neurais "morrem" até os anos 2000.

🔥 2010–2019 — Deep Learning Explode

2012 · AlexNet / ImageNet

Rede neural profunda (CNN) esmaga a competição de visão computacional por 10+ pontos percentuais. Marca o renascimento do deep learning. GPUs + dados + profundidade = mágica.

2013 · Word2Vec

Google descobre que palavras podem virar vetores numéricos com relações semânticas: rei - homem + mulher = rainha. Revoluciona NLP — de regras para representações aprendidas.

2017 · Transformer — "Attention Is All You Need"

Google publica a arquitetura que mudou tudo. Processa texto em paralelo (não sequencial como RNNs), escala melhor, e o mecanismo de atenção permite capturar dependências longas. Base de GPT, BERT, Claude, Gemini — tudo.

2018–2019 · GPT-1, BERT, GPT-2

OpenAI e Google mostram que pré-treinar em texto massivo + fine-tuning funciona absurdamente bem. GPT-2 (1.5B params) gera texto tão bom que eles "têm medo" de publicar. Scaling laws começam a se confirmar.

💡 Lição: Não houve "eureka" — a mesma ideia (redes neurais) + mais dados + mais compute + melhores arquiteturas = revolução.

🚀 2020–2023 — A Explosão Generativa

2020 · GPT-3 (175B parâmetros)

OpenAI escala 100x e descobre "emergent abilities" — o modelo sabe fazer coisas que ninguém ensinou explicitamente. Few-shot prompting funciona. A API abre acesso a startups.

Nov/2022 · ChatGPT

GPT-3.5 + RLHF + interface de chat = 1M usuários em 5 dias. Não era um modelo novo — era a mesma IA agora acessível. O mundo descobre que pode conversar com IA. Tudo muda.

2022 · ReAct + Chain-of-Thought

Pesquisadores descobrem que LLMs raciocinam melhor se você pedir "pense passo a passo". ReAct combina raciocínio + ação em loop — nasce o padrão que define agentes modernos.

Jun/2023 · Function Calling (OpenAI)

OpenAI lança tool use nativo: o modelo retorna JSON estruturado dizendo qual função chamar e com quais parâmetros. Agentes deixam de ser hack → viram produto de primeira classe.

🌟 2024–2025 — A Era dos Agentes

2024 · Claude + MCP (Anthropic)

Anthropic lança o Model Context Protocol — padrão aberto para agentes se conectarem a qualquer ferramenta (banco, APIs, arquivos). Agentes ganham um "USB universal". Microsoft, Google, OpenAI adotam.

2024 · Cursor / Windsurf / Devin

IDEs com agentes que escrevem código, rodam testes, fazem PRs. Devin (Cognition) é o primeiro "engenheiro de software AI" autônomo. Programadores passam de escrever código para supervisionar agentes.

2024 · Computer Use (Claude)

Agentes que controlam mouse e teclado, navegam sites, preenchem formulários. Não precisam de API — usam interfaces como humanos. Qualquer software vira "ferramenta" do agente.

2025 · Multi-agentes em produção

Empresas rodando times de agentes especializados: um pesquisa, outro analisa, outro escreve, outro revisa. Orquestradores como CrewAI, AutoGen, LangGraph coordenam "equipes de IAs".

💡 De 1950 a 2020: modelos que entendem. De 2020 em diante: modelos que AGEM.

E o ChatGPT?

30/nov/2022
0usuários em 5 dias

O que era, exatamente?

  • GPT-3.5 — LLM de ~175B parâmetros (já existia como API desde 2020)
  • RLHF — humanos avaliaram 100k+ respostas para ensinar "o que é útil"
  • Interface de chat — qualquer pessoa podia conversar com a IA, sem saber programar
A revolução foi de acesso, não de modelo. A tecnologia já existia — faltava a UX.
